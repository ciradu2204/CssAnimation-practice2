      About This Class

      Hey! Ever wanted to build your own interactive website, app, or game? Become a white-belt code-ninja in this fun introduction class to JavaScript. You’ll learn the basics of the JavaScript language, and be able to add logic, interaction and animation to your web projects. You’ll see how JavaScript interacts with HTML and CSS. By the end of the class you’ll be able to make your browser do all kinds of fun stuff – and you’ll be on your way to making games, animating web elements, and making interactive apps!

      This class is for anyone who likes apps, websites, design, or DIY. You’ll get to understand a fundamental part of the modern web, and be able to boast to your friends of your hacker-ways by the end of it! You'll also leanr a little bit about CSS and HTML!

      You don’t need to know any JavaScript, but some very basic knowledge of HTML and CSS will help.
